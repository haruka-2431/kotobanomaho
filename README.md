# ことばの魔法

気分に合わせて名言を表示するWebアプリ。  
GSAPによる滑らかなアニメーションとテーマ切り替え機能を実装。

**🌐 デモ:** https://haruka-2431.github.io/kotobanomaho/

## 主な機能

- 気分別の名言表示（やる気、自信、落ち込み、気づき）
- テーマ切り替え（通常モード/恋愛モード）
- GSAPによる滑らかなアニメーション

## 技術スタック

`JavaScript` `GSAP` `TailwindCSS` `DaisyUI`

## 工夫した点

- **GSAPアニメーション**  
  オープニング演出と名言表示に滑らかなアニメーションを実装。

- **テーマ切り替え機能**  
  通常モード・恋愛モードで表示内容と背景色を動的に変更。

- **レスポンシブ対応**  
  スマホ・タブレット・PCで快適に動作。
